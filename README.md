# How to install
1. Clone repo
2. Install library
`
python3 -m pip install bs4
`
# How To Use
1. Edit URLs in main.py
```python
# Current example
URLs = [
    'https://www.tokopedia.com/istorebdg/macbook-air-m2-chip-2022-13-inch-8gb-256gb-garansi-resmi-ibox-inter-silver-7c6c',
    ...
]
```
2. Run python
`
python3 src/main.py
`
3. Example output
